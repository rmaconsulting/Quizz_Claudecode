# Quiz: Verdadeiro ou Falso — Logística e Supply Chain

Aplicação web de página única para estudo de conceitos de logística
e supply chain. 30 questões divididas em três níveis, com feedback
e explicação imediata a cada resposta.

**Demo:** https://quiz-logistica-supply-chain.vercel.app/

## Sobre

Projeto pessoal de estudo, construído para praticar React e Vite e,
ao mesmo tempo, revisar conteúdo de supply chain. Totalmente
client-side, sem backend e sem persistência: cada sessão é
independente.

## Funcionalidades

- Três níveis de dificuldade, mais modo misto com todas as questões
- Temporizador por questão, com timeout contando como erro
- Explicação exibida logo após cada resposta
- Tela de resultado com desempenho geral e por nível

## Stack

React 18, Vite, JavaScript, CSS. Deploy na Vercel.

## Rodando localmente

npm install
npm run dev
